Pengantar GDScript

GDScript adalah bahasa pemrograman yang digunakan di Godot Engine. Ini seperti Python, tapi khusus untuk membuat game. GDScript mudah dipelajari karena sintaksnya sederhana.

Untuk mulai, buat script baru di Godot dan tulis kode di sana. Contoh sederhana:

extends Node

func _ready():
print("Halo, dunia!")

Variabel dan Tipe Data

Variabel adalah tempat menyimpan data. Di GDScript, kita pakai var untuk mendeklarasikan variabel. Tipe data seperti int (bilangan bulat), float (bilangan desimal), String (teks), bool (benar/salah).

Contoh:

var nama = "John" # String
var umur = 25 # int
var tinggi = 1.75 # float
var aktif = true # bool

Operator

Operator digunakan untuk melakukan operasi matematika atau logika. Misalnya + untuk tambah, - untuk kurang, == untuk bandingkan sama.

Contoh:

var a = 5
var b = 3
var hasil = a + b # hasil = 8
if a > b:
print("a lebih besar")

Kontrol Aliran

Kontrol aliran seperti if-else untuk memutuskan apa yang dilakukan, dan loop seperti for atau while untuk mengulang.

Contoh if:

if umur >= 18:
print("Dewasa")
else:
print("Anak-anak")

Contoh loop:

for i in range(5):
print(i) # Cetak 0 sampai 4

Fungsi

Fungsi adalah blok kode yang bisa dipanggil berulang. Gunakan func untuk membuatnya.

Contoh:

func sapa(nama):
return "Halo, " + nama

print(sapa("Alice")) # Cetak "Halo, Alice"

Array

Array adalah daftar nilai. Di GDScript, array bisa menyimpan berbagai tipe data.

Contoh:

var buah = ["apel", "pisang", "jeruk"]
print(buah[0]) # Cetak "apel"
buah.append("mangga") # Tambah item

Dictionary

Dictionary adalah struktur data yang menyimpan pasangan kunci-nilai. Berguna untuk data seperti nama dan nilai.

Contoh:

var orang = {"nama": "John", "umur": 30}
print(orang["nama"]) # Cetak "John"
orang["pekerjaan"] = "Programmer" # Tambah kunci baru

Class dan Object

Class adalah blueprint untuk membuat object. Di GDScript, gunakan class untuk mendefinisikan.

Contoh:

class Mobil:
var warna
func _init(w):
warna = w

var mobil1 = Mobil.new("merah")
print(mobil1.warna) # Cetak "merah"

Node dan Scene

Di Godot, segalanya adalah Node. Scene adalah kumpulan node. Gunakan get_node() untuk akses node lain.

Contoh:

extends Node2D

func _ready():
var label = get_node("Label")
label.text = "Halo dari script!"

Input dan Output

Input dari pemain pakai Input.is_action_pressed(). Output pakai print() atau ubah UI.

Contoh:

func _process(delta):
if Input.is_action_pressed("ui_right"):
print("Gerak kanan")

Topik Lanjutan

Topik lanjutan termasuk signals, inheritance, dan export. Signals untuk komunikasi antar node.

Contoh signals:

signal skor_berubah

func tambah_skor():
emit_signal("skor_berubah")
